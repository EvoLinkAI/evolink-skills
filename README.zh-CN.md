# EvoLink Agent Skills

**一个 API Key，所有 AI 模型。** 让你的 AI 智能体拥有多模态能力。

兼容 [OpenClaw](https://github.com/openclaw/openclaw)、[Claude  Code](https://claude.ai/code)、[Cursor](https://cursor.com)、[Gemini CLI](https://github.com/google-gemini/gemini-cli) 等所有支持 [Agent Skills](https://agentskills.io) 的产品。

## 安装

```bash
clawhub install cheapest-image
```

## 配置

在 [evolink.ai](https://evolink.ai) 获取 Key：

```bash
export EVOLINK_API_KEY="your-key"
```

## 可用 Skills

| Skill | 功能 | 价格 |
|-------|------|------|
| [cheapest-image](skills/cheapest-image/) | 可能是最便宜的 AI 生图 | ~¥0.026/张 |

更多 Skill 即将推出——视频、音乐、多模型路由。
